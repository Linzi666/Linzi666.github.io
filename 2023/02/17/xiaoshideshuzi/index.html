消失的数字

难度： 简单

题目描述：

数组nums包含从0到n的所有整数，但其中缺了一个。请编写代码找出那个缺失的整数。你有办法在O(n)时间内完成吗？

示例1

输入：[3,0,1]
输出：2

示例2

输入：[9,6,4,2,3,5,7,0,1]
输出：8

解题思路：

两个相同的数字异或的结果是0，而0和任何数异或，都是它本身。所以我们只需要把1-数组的长度和数组的元素依次异或。最后得到的值就是消失的数字。

假设：有数组 [3,0,1]

1 - 数组的长度：1，2，3

把这6个数一起异或后，得到2是消失的数字。

代码：

    int missingNumber(vector<int>& nums) {
        int len = nums.size();
        int ret = 0;
        for(int i = 1; i <= len ; i++)
        {
            //异或
            ret = ret ^ nums[i-1] ^ i;
        }
        return ret;
    }
